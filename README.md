# Subject Manager by Ivan Sosniuk

Solution було створено в JetBrains Rider на Enveavor OS
Версія dotnet - 10

## Користування
`lessons` -> виводить заняття

`subjects` -> виводить предмети з індексами -> приймає індекс і виводить за ним повну інформацію предмета з його заняттями -> знову прийматиме індекс до введення `back` 

`exit` -> вихід з програми 

## Структура
Subject_Manager/Program.cs - enrty point, наразі консольний застосунок 
### Model
Entity - класи зберігання даних

View - класи відображення, створення та редагування. У об'єктах View для відображення та редагування GUID присутній, для створення - відсутній

Enums - переліки (не винесені в окремий проєкт з огляду на те що їх всього 2)
### Storage
PrimitiveStorage - статичий клас-контейнер списків Entity, поки без серіалізації з захардкодженими тестовими даними

Repository - статичні класи для операцій зі списками з PrimitiveStorage. Також парсять Entity в View і навпаки. 
### Services
Service - класи для CRUD операцій з даними. Наразі готові ендпоінти Create, Update, GetAll, GetById, валідація Id для Create, GetById
### Utils
Utils - статичні методи-утиліти
